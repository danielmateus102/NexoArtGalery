# NexoArtGalery

**ArtGaleryWebsite**

Este repositorio contiene el prototipo de la página web desarrollada para **NEXO Art Room**, concebido como una herramienta de validación temprana mediante la técnica de *puerta falsa*. La plataforma permite a los usuarios comparar distintas opciones de espacios creativos, explorar recorridos visuales en 360° y registrar preferencias reales de navegación. El objetivo principal es medir el interés de artistas y público general antes de la construcción física del proyecto.
